# Shuttle to Overhead

Three escalating rounds of a run and max-rep jerks in the time remaining, the run getting longer each round.

**0:00-2:00**
**For Time**
  Run 400m
  max Jerk (200lb/155lb)
  *rest 1 min*

**3:00-6:00**
**For Time**
  Run 600m
  max Jerk (200lb/155lb)
  *rest 2 min*

**8:00-12:00**
**For Time**
  Run 800m
  max Jerk (200lb/155lb)

*Type: Multi Part · Category: Games*
