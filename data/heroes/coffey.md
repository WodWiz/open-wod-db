# Coffey

A descending back squat and bench press ladder (50-35-20) with an 800m run before each set, finishing with a single muscle-up after the last run.

**For Time**
  Run 800m

  50 Bench Press (135lb/95lb)
  Run 800m
  35 Back Squat (135lb/95lb)
  35 Bench Press (135lb/95lb)
  Run 800m
  20 Back Squat (135lb/95lb)
  20 Bench Press (135lb/95lb)
  Run 800m
  1 Muscle Up
