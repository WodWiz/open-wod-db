# Hope

Five one-minute stations — burpees, power snatches, box jumps, thrusters, and chest-to-bar pull-ups — repeated for three rounds with a minute of rest between rounds, following the same station format as Fight Gone Bad.

**3 rounds for time**
  Burpee
  Power Snatch (75lb/75lb)
  Box Jump
  Thruster (75lb/75lb)
  Chest To Bar Pull Up

*Type: Interval · Category: Girl*
